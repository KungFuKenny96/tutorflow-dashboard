📧 Email Notifications

Get emails for new sessions, payments, and a daily morning summary. Uses EmailJS — free, no backend needed.

⚡ Quick Setup (3 steps, ~5 min)
1. Go to emailjs.com → Sign up free → Create a service (Gmail)
2. Create an email template (see template IDs below)
3. Paste your keys here and click Save
Template IDs — create one per notification type
▶ How to create the email templates on EmailJS
In each template, use these variables:

📅 New Session template:
Subject: New class scheduled — {{student_name}}
Body variables: {{student_name}} {{date}} {{time}} {{subject}} {{duration}} {{tutor_name}}

💰 Payment template:
Subject: Payment received — {{student_name}}
Body variables: {{student_name}} {{amount}} {{method}} {{date}} {{balance}}

🌅 Daily Summary template:
Subject: TutorFlow Daily — {{date}}
Body variables: {{date}} {{sessions_today}} {{sessions_list}} {{total_students}} {{monthly_revenue}} {{outstanding_dues}}

💡 Tip: Set "To Email" field in EmailJS template to {{to_email}}

Manage your locally stored data.

📋 One-Time Setup Guide

Follow these 6 steps to connect Google Sheets as a free backend:

Go to sheets.google.com and create a new blank spreadsheet. Name it "TutorFlow Data".
In the spreadsheet, click Extensions → Apps Script. Delete any existing code.
Paste the entire contents of google-apps-script.gs (included in your download) into the editor. Save it.
Select setupSheets in the dropdown and click Run. Approve permissions when asked.
Click Deploy → New Deployment → Web App. Set "Execute as: Me", "Who has access: Anyone". Deploy.
Copy the URL, paste it above, click Save & Test Connection. Done!
